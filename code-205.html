205 Restablecer Contenido: Similar a un código 204, significa que el servidor ha procesado la solicitud pero no va a devolver ningún contenido. Sin embargo, también requiere que tu navegador restablezca la vista del documento.

205 Reset Content: Similar to 204, there is no content sent for this request. But, it tells the user agent to reset the document which sent this request.
